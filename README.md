#### News AI Agents Using CrewAI And Google Gemini Pro LLM Model

## In this project, CrewAI and the Google Gemini Pro LLM model are used to create an intelligent application where multiple AI agents communicate and collaborate to generate insightful and comprehensive blog posts. These agents are designed to focus on trending news topics

## CrewAI tools empower agents with capabilities ranging from web searching and data analysis to collaboration and delegating tasks among coworkers

## Gemini Pro is a Large Language Model (LLM) developed by Google
